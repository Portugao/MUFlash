INSTALLATIONSANLEITUNG
----------------------
1) Kopieren Sie MUFlash in das Verzeichnis modules.
2) Initialisieren und aktivieren Sie MUFlash in der Modulverwaltung.
3) Setzen Sie das Verzeichnis `userdata/MUFlash/` inklusive aller Unterordner auf beschreibbar.
